## 📋 人体检测系统分析报告

### **一、系统主要功能**

这是一个**STM32F103C8 + MAX30102 + DS18B20**的**智能健康监测系统**，主要功能包括：

| 功能           | 说明                       |
| :------------- | :------------------------- |
| 🫀 **心率测量** | MAX30102 PPG传感器实时采集 |
| 🫁 **血氧测量** | MAX30102红外LED反射计算    |
| 🌡️ **体温测量** | DS18B20数字温度传感器      |
| 📊 **实时显示** | OLED屏幕显示数据           |
| ⏰ **数据监测** | 三层报警阈值设置           |
| 📡 **蓝牙控制** | ESP8266通过UART支持遥控    |
| 🔔 **告警提示** | BEEP蜂鸣器 + LED指示灯     |

### **二、核心引脚配置**

#### **传感器接口**

```
MAX30102 (心率/血氧)
├─ SCL: PA11  (I2C软件模拟)
└─ SDA: PA8   (I2C软件模拟)

DS18B20 (体温)
└─ DQ:  PB12  (单总线协议)

ADC (备用)
└─ CH0: PA0   (10位ADC采样)
```

#### **外设控制**

```
PB0   - LED 指示灯
PB1   - BEEP 蜂鸣器
PB14  - RELAY_1 (热水控制)
PB15  - RELAY_2 (温控制)
PB13  - MOTOR_A1 (电机)
PC13  - LED_SYS (系统LED)
```

### **三、数据采集周期分析**

#### **定时器配置**

```c
TIM2_Init(99, 7199);  // main.c:72行
```

**计算过程：**

- STM32F103 APB1时钟：72 MHz
- 分频系数：PSC = 7199
- 自动重装值：ARR = 99
- **中断频率** = 72MHz / (7199+1) / (99+1) = **100 Hz**
- **中断周期** = 1/100 = **10 ms**

#### **数据采集频率**

```c
// main.c:258行 - Monitor_function中
if(time_num % 10 == 0 && system_state != 2)
{
    temp_value = DS18B20_Get_Temp();      // 体温
    heart_rate = get_real_spo2();         // 心率  
    spo2 = get_real_heart_rate();         // 血氧
}
```

**采集周期计算：**

- `time_num` 每个主循环增加1，受中断驱动
- 条件：`time_num % 10 == 0`
- **采集间隔** = 10 × 10ms = **100 ms** (即 **10 Hz**)

#### **数据上传频率**

```c
// main.c:291行 - 上传到串口
if(time_num % 30 == 0 && system_state != 2)
{
    UsartPrintf(USART1, "温度:%d.%d\r\n", temp_value/10, temp_value%10);
    // ... 上传心率血氧
}
```

- **上传周期** = 30 × 10ms = **300 ms** (即 **3.3 Hz**)

### **七、主循环执行序列**

```c
int main(void)
{
    // 1. 系统初始化
    Gpio_Init();           // GPIO初始化
    Oled_Init();          // OLED屏幕
    Usart1_Init(9600);    // 串口 (蓝牙通讯)
    TIM2_Init(99,7199);   // 定时器 (100Hz中断)
    DS18B20_Init();       // 体温传感器
    max30102_init();      // 心率血氧传感器
    
    while(1)  // 每循环约100ms
    {
        // 2. 采集数据 (每100ms)
        temp_value = DS18B20_Get_Temp();
        heart_rate = get_real_spo2();
        spo2 = get_real_heart_rate();
        
        // 3. 处理按键输入
        Key_function();        
        
        // 4. 数据监测与上传 (每300ms上传一次)
        Monitor_function();   
        
        // 5. 报警判断
        Alarm_function();     
        
        // 6. OLED显示更新
        Display_function();   
        
        time_num++;  // 计时器递增 (由TIM2中断驱动)
    }
}
```
